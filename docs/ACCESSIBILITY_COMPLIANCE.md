# Accessibility Compliance

推理ゲームMVPのアクセシビリティ基準です（目標: WCAG 2.1 AA）。

## 必須要件

- すべての操作要素（`button` / `select`）はキーボード操作可能
- `:focus-visible` が視認できる状態を保つ
- 正解/不正解は色だけでなくテキストでも伝える
- 見出し階層を保つ（`h1` → `h2`）

## MVPでの確認項目

- [ ] 「ゲーム開始」「質問実行」「解答する」がTab移動で操作できる
- [ ] 選択UI（犯人/凶器/現場）がキーボードで変更できる
- [ ] CPU思考中にユーザーが誤操作しないようボタンが適切に無効化される
- [ ] ログの文言が「持っている/持っていない」で明示される
